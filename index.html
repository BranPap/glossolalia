Welcome to Glossolalia!

Glossolalia is a free, open-source language exercise app built with jsPsych. No frills, no fees, no fucking ads; just the most barebones language practice you can think of!

Please note that this project is a labour of love, so updates are likely to be irregular and the offering of languages will, of course, be small to start off.
